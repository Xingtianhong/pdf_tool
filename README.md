# pdf_tool
处理pdf文件(拆分、合并、加页码、重排页码）

# PDF 工具箱

## 主菜单

- **1.** 去除原页码并添加新页码（覆盖图片 + 底部居中）
- **2.** 拆分 PDF（按页码范围）
- **3.** 合并多个 PDF
- **4.** 插入另一个 PDF 并重编页码（右下角）
- **0.** 退出

---
